# Personal Expense Tracker Dashboard
An interactive **Personal Expense Tracker Dashboard** built using Microsoft Excel to analyze expenses by category, payment mode, and month.

## Features
- Dynamic KPI Cards
- Interactive Month Slicer
- Expense by Category Analysis
- Monthly Expense Trend Chart
- Payment Mode Distribution Chart
- Pivot Tables and Pivot Charts
- Professional Dashboard Design

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- XLOOKUP
- Excel Tables
- Data Visualization
